## Dictionaries

A console application for managing multilingual dictionaries with words and translations.

### Features

- Create dictionaries with a custom language pair (e.g. ENG-UKR)
- Store and load dictionaries using JSON serialization
- Manage multiple dictionaries in memory
- Add, replace, and remove words
- Add, replace, and remove translations
- Search for translations of a word
- Export a word with its translations
- Menu-driven console interface

### Key Concepts

- Dictionaries and collections
- JSON serialization and deserialization
- Separation of responsibilities (menus, services, storage)
- input validation

## Quiz Application

A console-based quiz system.

### Features

- User registration and login
- Quiz selection or mixed quiz mode
- Support for questions with multiple correct answers
- Score calculation and result saving via JSON
- View personal results
- View Top 20 results
- Quiz editor
- Settings menu

### Key Concepts

- JSON-based data persistence
- User, quiz, and result management
- LINQ and collections
- Console navigation and menus
