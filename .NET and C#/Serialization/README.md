# Serialization — Magazine Project

A simple C# console project demonstrating **JSON serialization**, **deserialization**, and working with nested objects.  
The project includes classes for `Magazine`, `Article`, and a serializer using `System.Text.Json`.

---

## Contents

1. **Article.cs** – represents a magazine article.  
2. **Magazine.cs** – magazine model with a list of articles.  
3. **MagazineSerializer.cs** – handles JSON serialization/deserialization.  
4. **Program.cs** – console input, saving to file, loading back, and printing info.
