# C# Projects

A collection of **C# console applications** created for learning and practice.  
The repository demonstrates solid knowledge of **C#/.NET**, **OOP**, **collections**, **LINQ**, **delegates & events**, **file handling**, and **design patterns**.

## Key Skills
- C# / .NET
- Object-Oriented Programming (OOP)
- Classes, Structs, Interfaces, Inheritance, Polymorphism
- Collections, Generics, Custom Iterators
- LINQ
- Delegates, Events, Lambda Expressions
- Operator Overloading, Indexers
- File I/O (txt), JSON Serialization
- IDisposable & Garbage Collection
- Design Patterns (Composite, Observer, Prototype)

## Projects Overview
- **Basics**: arrays, strings, arithmetic, console input/output  
- **OOP**: abstract classes, interfaces, polymorphism  
- **Collections & Generics**: filtering, iteration, reusable logic  
- **Delegates & Events**: event-driven models (bank card, backpack)  
- **LINQ**: complex queries over nested collections  
- **File & JSON**: quizzes, dictionaries, magazines, poem storage  
- **Applied Apps**: warehouse, employees, quizzes, generators
