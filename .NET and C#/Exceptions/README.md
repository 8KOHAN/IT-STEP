# Class Money

A simple C# console application demonstrating operator overloading and exception handling with a custom `Money` class.

## Features
- Stores money in dollars and cents  
- Supports operator overloading: `+`, `-`, `*`, `/`, `++`, `--`, `<`, `>`, `==`, `!=`  
- Prevents negative balances using a custom exception `BankruptException`  
- Demonstrates usage through console output
